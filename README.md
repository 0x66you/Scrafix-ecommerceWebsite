### 作品說明
Scrafix 是我在2021年11月初花兩個多禮拜完成的購物網站。<br>
我的目標是利用vue框架寫出一個**分工明確**的網頁，每個vue元件要可以被**重複利用**，<br>
或是有某個**特定功能**因此必須獨立出來一個.js檔。<br>

* 首頁macbook隨滾輪捲動開合，頁首頁尾字體淡入與淡出
* RWD設計，隨頁面大小不同版面配置
